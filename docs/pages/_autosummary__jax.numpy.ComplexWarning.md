# jax.numpy.ComplexWarning[\#](#jax-numpy-complexwarning "Link to this heading")

*exception* jax.numpy.ComplexWarning[\#](#jax.numpy.ComplexWarning "Link to this definition")  
The warning raised when casting a complex dtype to a real dtype.

As implemented, casting a complex number to a real discards its imaginary part, but this behavior may not be what the user actually wants.
