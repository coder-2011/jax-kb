# Getting Started with JAX[\#](#getting-started-with-jax "Link to this heading")

Welcome to JAX! The JAX documentation contains a number of useful resources for getting started. [Quickstart: How to think in JAX](notebooks/thinking_in_jax.html) is the easiest place to jump in and get an overview of the JAX project, its execution model, and differences with NumPy.

If you’re starting to explore JAX, you might also find the following resources helpful:

- [Key concepts](key-concepts.html) introduces the key concepts of JAX, such as transformations, tracing, jaxprs and pytrees.

- [🔪 JAX - The Sharp Bits 🔪](notebooks/Common_Gotchas_in_JAX.html) lists some of JAX’s sharp corners.

- [Frequently asked questions (FAQ)](faq.html) answers some frequent JAX questions.

## Building on JAX[\#](#building-on-jax "Link to this heading")

JAX provides the core numerical computing primitives for a number of tools developed by the larger community. For example, if you’re interested in using JAX for training neural networks, two well-supported options are [Flax](https://flax.readthedocs.io/) and [Haiku](https://dm-haiku.readthedocs.io/).

For a community-curated list of JAX-related projects across a wide set of domains, check out [Awesome JAX](https://github.com/n2cholas/awesome-jax).
