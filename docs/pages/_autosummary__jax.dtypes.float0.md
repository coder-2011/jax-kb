VoidDType(length, /) –

DType class corresponding to the numpy.void scalar type.

Warning

`np.dtypes.VoidDType` cannot be instantiated directly. Use `np.dtype("void[{unit}]")` instead.

See numpy.dtype for the typical way to create dtype instances and [Data type objects (dtype)](https://numpy.org/doc/stable/reference/arrays.dtypes.html#arrays-dtypes "(in NumPy v2.4)") for additional information.
