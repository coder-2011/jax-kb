Alias of [`numpy.set_printoptions()`](https://numpy.org/doc/stable/reference/generated/numpy.set_printoptions.html#numpy.set_printoptions "(in NumPy v2.4)").

JAX arrays are printed via NumPy, so NumPy’s printoptions configurations will apply to printed JAX arrays.

See the [`numpy.set_printoptions()`](https://numpy.org/doc/stable/reference/generated/numpy.set_printoptions.html#numpy.set_printoptions "(in NumPy v2.4)") documentation for details on the available options and their meanings.
